## Server folder
>npm init -y
>npm install i express
>npm i -g nodemon
>npm i dotenv
>npm i mongoose 

## Run the server
>npm run dev

## API Endpoint
>GET    /WorkOuts        -->GET all the Workout docs
>POST   /WorkOuts        -->CREATE a new workout doc
>GET    /Workouts/:id    -->GET a single workout doc
DELETE  /Workouts/:id    -->DELETE a single workout doc
PATCH   /worlouts/:id    -->Update a single workout

## FrontEnd
>npx create-react-app frontend (create a folder frontend)
>npm start
